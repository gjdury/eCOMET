# Add feature_info to an existing mmo from a full_feature CSV

Add feature_info to an existing mmo from a full_feature CSV

## Usage

``` r
AddFeatureInfo(mmo, full_feature_csv)
```

## Arguments

- mmo:

  An existing mmo list (will be returned with \$feature_info added)

- full_feature_csv:

  Path to the MZmine full feature table (CSV)

## Value

The same mmo with mmo\$feature_info set
