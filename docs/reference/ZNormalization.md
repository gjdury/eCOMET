# Z-normalize the peak area in the mmo object

This function applies Z-score normalization to the peak area in the
feature data of the mmo object.

## Usage

``` r
ZNormalization(mmo)
```

## Arguments

- mmo:

  The mmo object

## Value

The mmo object with Z-normalized feature data (mmo\$zscore)

## Examples

``` r
if (FALSE) {
mmo <- ZNormalization(mmo)
}
```
